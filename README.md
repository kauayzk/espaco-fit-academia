# Espaço Fit Academia

Landing page de alta conversão criada para a **Espaço Fit Academia**, em São José da Mata — PB. O projeto combina identidade visual dark, navegação fluida e uma jornada de matrícula integrada ao WhatsApp.

[**Acessar o site ao vivo →**](https://espaco-fit-academia-sjm.vercel.app/)

![Prévia da landing page Espaço Fit](./public/og.png)

## Sobre o projeto

O site foi desenvolvido para transformar visitas em contatos qualificados e, ao mesmo tempo, facilitar o atendimento da academia. A experiência é responsiva, rápida e pensada para celular, onde acontece a maior parte das conversões.

### Principais recursos

- Landing page responsiva com identidade visual própria da academia
- Mega menu, animações e transições suaves durante a navegação
- Apresentação de estrutura, horários, planos e diferenciais
- Fluxo guiado de matrícula com validação dos dados
- Mensagem de WhatsApp gerada automaticamente para a equipe
- Estrutura preparada para captação de leads com Cloudflare D1 e Drizzle ORM
- Painel protegido para acompanhamento, filtros e exportação de contatos
- Eventos de conversão e página de privacidade
- SEO, Open Graph e dados estruturados para busca local

## Tecnologias

- TypeScript
- React 19 e Next.js 16
- vinext e Vite
- Cloudflare Workers e D1
- Drizzle ORM
- CSS responsivo com animações nativas

## Executando localmente

Requer Node.js `22.13.0` ou superior.

```bash
npm install
npm run dev
```

Para validar a versão de produção:

```bash
npm test
```

Variáveis de ambiente e credenciais não fazem parte do repositório. A área de gestão depende da configuração de usuários autorizados no ambiente de hospedagem.
